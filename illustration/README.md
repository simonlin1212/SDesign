# 配图 · Illustration(SDesign 子能力)

给中文正文配一张**手绘风插画**——把一个判断 / 流程 / 状态 / 隐喻画出来。**去 AI 味最锋利的一刀**:满屏精致 AI 配图本身就是 AI 味重灾区,手绘草图反其道而行。

- 用法:打开 [`prompt.md`](prompt.md),复制提示词、替换变量,粘给任意图像 AI(Gemini / Nano Banana / …)。
- 这是和 `aesthetics/`(整页排版的设计系统)**并列的一条线**,不是第 7 个美学家族——它解的是"正文里插什么图",不是"页面长什么样"。

> 方法论吸收自 [Ian Xiaohei Illustrations](https://github.com/helloianneo)(MIT)。**SDesign 只吸收了"手绘去 AI 味"的方法论,未使用其 IP 角色「小黑」**;致谢见根目录 [`NOTICE`](../NOTICE)。

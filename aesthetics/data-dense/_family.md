# 数据密集 / Data Dense

图表/表格/指标为主 + 紧间距 + 暗色仪表盘;专业、信息高密度。

**本美学特别说明**:信息高密度是对的——一屏多数据点/图表/表格正常,别强行留白;CTA、按钮可多放。 玻璃卡技法:真毛玻璃 `backdrop-filter:blur(20px)`(浮起层 40px)+ 半透白填充 `rgba(255,255,255,.08~.12)` + 1px 白边 `rgba(255,255,255,.18)` + 内高光只走顶+左两条边(单光源)+ 柔投影 `0 8px 32px rgba(0,0,0,.3)`;玻璃上字重加一档、小标签加 text-shadow 保可读。

**代表设计系统**:Binance · ClickHouse · GitHub

> 美学家族 = 浏览/分类单位。进来挑一个设计系统,复制它的 `prompt.md` 即可。

# 科技暗黑 / Tech Dark

深色底 + 高对比 + 无衬线 + 单一霓虹/品牌强调色;冷峻精密,开发者/SaaS 气质(含终端等宽)。

**本美学特别说明**:深色为底、强调色用得稀、网格精密对齐;可有极克制的微光,别滥用发光。

**代表设计系统**:Spotify · MongoDB · OpenCode · HashiCorp · Discord · Sentry · Supabase · Shopify · Perplexity · Nvidia · Linear · Resend · Framer · xAI · Warp

> 美学家族 = 浏览/分类单位。进来挑一个设计系统,复制它的 `prompt.md` 即可。
